# AI Factory Architecture

AI Factory 是一个自动化 AI 工程平台。

目标：
- 自动生成 React 项目
- 自动 Docker 化
- 自动端口管理
- 自动 compose
- 自动日志
- 自动 Agent 初始化

核心理念：
- 所有项目统一管理
- 所有项目自动隔离
- 所有项目标准化生成
- AI 参与开发流程
- 自动化优先

技术栈：
- Bash
- Docker
- Docker Compose
- React
- Vite
- Nginx
- Git
- Node.js

目录结构：

AI-Factory/
├── projects/
├── templates/
├── infra/
├── scripts/
├── logs/
└── runtime/

V1 已完成
